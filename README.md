## ğŸš€ How to Use

1. **Open the Application**
   - Simply open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari)
   - **Note:** An internet connection is required to fetch recipes from the API

2. **Search by Dish Name**
   - Click on the "Search by Dish" tab
   - Enter the name of a dish (e.g., "Pasta", "Chicken", "Pancakes", "Pizza")
   - Click "Search" or press Enter
   - View multiple recipe results with images, ingredients, and instructions

3. **Find Recipes by Ingredients**
   - Click on the "Find Recipes by Ingredients" tab
   - Enter an ingredient and click "Add" or press Enter
   - Add multiple ingredients you have available
   - Click "Find Recipes" to see suggested dishes
   - Recipes are sorted by match percentage (how many ingredients you have)
   - Each recipe shows which ingredients you have and which are missing
